# Заголовок проекта

TestUNIT

## Начало работы

Открыть VisualStudio
Выбрать папку с проектом 
Запустить нажав на папку правой кнопкой и выбрать от имени VisualStudio

### Необходимые условия

Процессор с тактовой частотой 1,8 ГГц или большей ОЗУ объёмом 2 ГБ
1-40 Гб доступного пространства на жестком диске 
Видеоадаптер поддерживающий разрешение экрана 1280 x 720 или выше

### Установка

1.Открыть браузер
2.Запустить главную форму Visual Studio в браузере 
3.Найти ссылку для скачивания Visual Studio
4.Скачать пакет для установки 
5.Открыть установщик 
6.Выбрать в инструментах модуль C# 
7.Загрузка закончена
